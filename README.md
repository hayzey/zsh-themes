# zsh-themes

A little collection of my zsh themes. 

## Usage

1. Copy the .zsh-theme file into `~/.oh-my-zsh/themes`
2. Update your `.zshrc` with the theme name (ie - `"return"`)
3. Run `source ~/.zshrc` (or open a new ZSH terminal).
